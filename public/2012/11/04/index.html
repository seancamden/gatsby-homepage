Samba on Raspberry Pi with Arch Linux

November 04, 2012

I followed Samba for Arch Linux and the Arch Linux Arm Guide to Samba to get Samba working on my Raspberry Pi.

This version of Arch Arm is using systemd to manage services, so there is no more /etc/rc.conf file to edit and no rc.d command available to start anything (as the Arch Linux Arm Guide instructs). One simply types systemctl start smbd.service to make it go (after configuring /etc/samba/smb.conf, of course).
